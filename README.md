 Problème
Comment aider une banque à détecter et réduire les fraudes de cartes de crédit grâce à la data ?

 Méthodologie
- Dataset : 1 000 transactions de cartes de crédit
- Nettoyage des données avec Power Query
- Création de mesures DAX (Nb Fraudes, Montant Fraudé, Taux de Fraude)
- Visualisation avec Power BI Desktop

 Solution
Dashboard interactif avec 2 filtres dynamiques (Merchant Name, State) permettant d'analyser les fraudes par mois, par région, par type de fraude et par niveau de risque.


 Impact business
- Identification des zones à risque (Uttar Pradesh en tête)
- Détection du pic de fraudes en septembre
- Les transactions E-commerce sont les plus exposées

 Recommandations
1. Renforcer la surveillance dans les États les plus touchés
2. Implémenter l'authentification à deux facteurs pour le E-commerce
3. Augmenter les contrôles en septembre

Outils utilisés
- Power BI Desktop
- Power Query
- DAX
